# My Userscripts

A monorepo for userscripts I’ve written for personal use, published under the MIT License.

## Scripts

| Script                                                            | Description                                                                                                                                                 |
| ----------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [**jetbrains-mono-everywhere**](./jetbrains-mono-everywhere/)     | Applies the JetBrains Mono font to text across all websites, with intelligent exclusions for icon fonts &c.                                                 |
| [**magnet-target-blank**](./magnet-target-blank/)                 | Opens magnet links in a new tab instead of navigating away from the current page.                                                                           |
| [**whatsapp-accents-workaround**](./whatsapp-accents-workaround/) | Fixes an issue on WhatsApp Web for Firefox on Linux where text starting with accented characters (entered using dead keys) gets deleted during composition. |
