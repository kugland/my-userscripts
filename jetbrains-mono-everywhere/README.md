# jetbrains-mono-everywhere

[![GitHub package.json version](https://img.shields.io/github/package-json/v/kugland/my-userscripts?filename=jetbrains-mono-everywhere%2Fpackage.json&label=github)](https://github.com/kugland/my-userscripts/blob/master/jetbrains-mono-everywhere)
[![Greasy Fork Version](https://img.shields.io/greasyfork/v/569423)](https://greasyfork.org/en/scripts/569423)
[![License: MIT](https://img.shields.io/github/license/kugland/my-userscripts)](../LICENSE)

A userscript that applies the JetBrains Mono font to text across all websites.

The script automatically replaces the font on any website with JetBrains Mono, while intelligently excluding icon fonts and elements that shouldn't be affected (such as Material Icons, Font Awesome, and other icon libraries).
